# MoodRing

Technica 2015 Hackaton winner for "Most Innovative Use of Web Technologies."

Filters news articles and graph trends based on time frame, topic, or tone.

## Technologies

- Node.js
- Meteor
- IBM Watson Tone Analyzer
- NPR API
- MongoDB
